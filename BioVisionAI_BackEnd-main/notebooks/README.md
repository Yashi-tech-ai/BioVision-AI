# Notebooks

Place Jupyter notebooks here for exploration and demos, e.g.:

- `01_data_exploration.ipynb` — Load and visualize HAM10000/ISIC samples
- `02_training_demo.ipynb` — Short training run and metrics
- `03_inference_and_gradcam.ipynb` — Single-image prediction and heatmap

Run from project root so that `import data`, `import models` etc. work (add `sys.path.insert(0, "..")` if needed).
